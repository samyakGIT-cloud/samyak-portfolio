## 🚀 How to Run Locally

### Method 1 – VS Code Live Server (Recommended)

1. Install [VS Code](https://code.visualstudio.com/)
2. Install the **Live Server** extension by Ritwick Dey
3. Open the `portfolio/` folder in VS Code
4. Right-click `index.html` → **"Open with Live Server"**
5. Portfolio opens at `http://127.0.0.1:5500`

### Method 2 – Direct File Open

1. Double-click `index.html`
2. Opens in your default browser (no server needed for viewing)

## ✏️ How to Customise

All editable fields are marked with `<!-- ✏️ -->` comments in `index.html`.

### Quick Edits

| What to change | Where to find it |
|---|---|
| Your name | `index.html` – hero title, navbar logo, footer |
| Email / Phone | `index.html` – contact section & `href` attributes |
| LinkedIn URL | `index.html` – all `href="https://www.linkedin.com/..."` |
| GitHub URL | `index.html` – all `href="https://github.com/..."` |
| Resume file | Replace `resume.pdf` in root folder; update `href="resume.pdf"` |
| Project GitHub links | `index.html` – project card anchor tags |
| Certificate links | `index.html` – `.btn-cert` anchor `href` attributes |
| Certifications details | `index.html` – cert cards, provider names, dates |
| Typing animation strings | `script.js` – the `strings` array (line ~60) |
| Accent colours | `style.css` – `:root` CSS variable block |

### Change Colour Theme

Edit these variables at the top of `style.css`:

```css
:root {
  --accent-cyan:   #00d4ff;   /* Primary accent */
  --accent-blue:   #3a7cff;   /* Secondary accent */
  --accent-violet: #8b5cf6;   /* Tertiary accent */
  --accent-green:  #10b981;   /* Status / success colour */
  --bg-primary:    #050d1a;   /* Main background */
}
```
